# Splashscreen
This plays the QuintOS splash screen on startup.

# How to install
Clone the git repo:
```sh
git clone https://github.com/snorain/splashscreen
```
cd into the main folder
```
cd splashscreen
```
Run the install script
```
bash install
```

# How to uninstall:
cd into the main folder
```
cd splashscreen
```
Run the uninstall script
```
bash uninstall
```
